------ Unity README -------

Die korrekte Unity Version ist 2019.3.2f

Damit alles korrekt funktioniert, bitte importieren Sie zusätzlich die benötigten Scripts aus dem "Shared Scripts" Repository.
Diese werden sowohl für den Unity Client als auch für den Game Server benötigt.
Für den Game Server sollten diese automatisch mitgepullt werden.
Wenn dies nicht der Fall ist, fügen Sie diese bitte auch dem Game Server hinzu.
